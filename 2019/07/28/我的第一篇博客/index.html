I LOVE U

发表于 2019-07-28 更新于 2020-02-01 分类于 分类测试

这就是我们共同的事

我萌生写博客的想法是因为你，我感觉到我好多东西都在自己肚子里面存着说不出来。我应该锻炼一下。跟你说的时候我只提到了找工作写博客是加分项。但那是最最最最浅显的帮助对我来说。我最初的目的是记录我和你的故事。把它当成咱们感情的见证。有事没事写点文字记录我们的生活。
我是有点自私。有时候的我只顾着我自己的学习。整天脑子里面除了学习就是玩。自从我对计算机着了迷我就迷失了自己。整天忍不住得盯着电脑看。自己的眼睛真的是很不争气。看完电脑之后我就感觉自己跟一个智障没什么区别。
